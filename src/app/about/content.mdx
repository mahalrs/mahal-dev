# About <br /> Rajwinder Mahal

Rajwinder Mahal is a skilled machine learning engineer specializing in machine learning and generative AI as an MS in Computer Science candidate at Columbia University. He holds a master's degree in Innovation and Entrepreneurship and a bachelor's degree in Computer Science from the University of California, Irvine.

Rajwinder has experience in building scalable machine learning systems and has worked on projects in natural language processing, computer vision, generative AI, and recommender systems. He has worked with a variety of machine learning frameworks, including TensorFlow, PyTorch, and Scikit-learn. He also has experience working with cloud computing platforms, including Google Cloud Platform and Amazon Web Services.

Rajwinder is deeply interested in the application of machine learning and state-of-the-art language models to solve real-world problems and making information more accessible and understandable. He is currently working on a generative AI experiment to transform the way people access news and make decisions while putting user privacy, trust, and preferences at the forefront.

Rajwinder also has a strong passion for open-source software and has contributed to several open-source projects, including [minikube](https://github.com/kubernetes/minikube), [Azure Orkestra](https://github.com/Azure/orkestra), and [HashiCorp Vault](https://github.com/hashicorp/vault). He loves to use open-source tools and is proficient in PyTorch, PyTorch Lightning, HuggingFace Transformers, Kubernetes, Docker, VS Code, Git, and Linux.

## Key Projects & Papers

### Multi-Modal Image Generation for News Stories

[Paper](/static/multi-modal-image-generation-for-news-stories.pdf)
&nbsp;&nbsp;
[GitHub](https://github.com/mahalrs/newsgen)

### News Summarization using Enhanced Content Features

[Paper](/static/news-summarization-using-enhanced-content-features.pdf)
&nbsp;&nbsp;
[GitHub](https://github.com/mahalrs/newsum)

This project explores the feasibility of using additional content features such as news headline, topic modeling, named entities, and sentiment analysis to generate targeted summaries that are concise and informative.

### Single Image Super Resolution

[Paper](/static/single-image-super-resolution.pdf)
&nbsp;&nbsp;
[GitHub](https://github.com/mahalrs/super-scaler)

Storing high-resolution images is a challenging problem due to its high storage cost and growing storage demands. To address this issue, this project explores the feasibility of using deep convolutional networks for efficient storage and on-demand high resolution image generation.

### Comparative Analysis of Machine Learning Models for Short-Term Rental Price Prediction

[Paper](/static/short-term-rental-price-prediction.pdf)
&nbsp;&nbsp;
[GitHub](https://github.com/mahalrs/airbnb-autopilot)

Short-term rentals are gaining popularity in recent years, with Airbnb leading the change. When listing properties, one of the biggest challenges is finding a competitive listing price. In this project, we conducted various experiments to perform comparative analysis of different machine learning models to predict listing price.
